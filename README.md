# Snapgram
Snapgram is a lightweight social media web app that lets users create profiles, share photos, and interact through likes, comments, and follows. Built with React, Vite, TypeScript, and Appwrite, it delivers a fast, clean, and responsive Instagram-style experience focused on simplicity and modern web performance.

---
📸 Snapgram

A modern social media web app for sharing moments, connecting with people, and engaging through photos and stories.

---
Overview

Snapgram is a lightweight social platform where users can:

* Create profiles
* Share photos and posts
* Follow other users
* Like and comment on content

It is designed to feel fast, minimal, and familiar-like a modern Instagram-style experience built with a clean developer-first architecture.

---
🚀 Features

* 👤 User authentication and profiles
* 📷 Photo upload and sharing
* ❤️ Like and comment system
* 👥 Follow/unfollow users
* 📰 Personalized feed
* ⚡ Fast and responsive UI
* 🔐 Secure backend integration (Appwrite)


  ---
🛠️ Tech Stack

* Frontend: React, Vite, TypeScript
* Styling: Tailwind CSS / CSS Modules
* Backend: Appwrite
* Deployment: Vercel


  ---
🧠 Inspiration

Snapgram was built to explore how modern social platforms work under the hood-focusing on:

* Clean UI/UX design
* Scalable frontend architecture
* Real-time social interactions
* Backend-as-a-service integration
